# FitnessProgressor – User Interface

This repository contains the user interface for the FitnessProgressor application.  
The UI is responsible for presenting workout data, insights, and recommendations to the user in a clear and intuitive way. It includes all screens required for the MVP, such as login, dashboard, workout logging, history, insights, and recommendations.

## Purpose
The UI serves as the primary interaction point for users. It enables them to:
- Log workouts
- View historical performance
- Receive insights and recommendations
- Manage their profile

## Repository Structure
- `/design` – Contains all UI design documents (personas, wireframes, storyboards, style guide)
- `/src` – Will contain UI implementation files in future sprints

## Status
This repository is currently in Sprint 1 setup. Development begins in Sprint 2.
